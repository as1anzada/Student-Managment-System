ax = b şəklində verilmiş tənliyə birdəyişənli standart xətti tənlik deyilir. Burada a ≠ 0. Bu tənliyin kökü x = b : a olur. 1. Tənliyin hər iki tərəfinə eyni ifadəni əlavə etmək və ya çıxmaq olar. 2. Tənliyin hər iki tərəfini sıfırdan fərqli hər hansı bir ədədə vurmaq və ya bölmək olar. Eynilik çevrilmələri ilə standart xətti tənliyə gətirilə bilən tənliyə xətti tənlik deyilir. Məsələn: ax + b = cx + d xətti tənlikdir. Standart xətti tənliyin həllinin varlıq şərtləri:
1. a ≠ 0 olarsa, yeganə x = b : a həlli vardır;
2. a = 0, b ≠ 0 olarsa, həll yoxdur: x = Ø;
3. a = 0, b = 0 olarsa, sonsuz sayda həlli var.
